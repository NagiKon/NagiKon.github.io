JavaScript - Basic JavaScript

發表於2021-09-09|更新於2021-09-11|JavaScript基礎

Semicolon

Statement 句尾必須加上分號。

var x = 3 * 7;
var f = function () { }; // function expr. inside var decl.

基本型態

  • booleans
  • numbers
  • strings
  • null: no object
  • undefined: no value

Null

Null 為 object 是一個 bug 但現在修改它，會導致舊的 code 出現問題因此不能修正。

console.log(typeof null) // object

Number

所有的 number 都是浮點數。

1 === 1.0 // true

// toString (all legal)
123..toString()
123 .toString()
123.0.toString()
(123).toString()

你可以使用 parseFloat 和 Number 把字串轉換成數值。

parseFloat 會在轉換之前，將輸入的值轉換成 String 後，才進行轉換。但相較於 Number 效能較快。

Number('123') // 123

parseFloat(true) // same as parseFlaot('true')
// NaN

Number(true)
// 1

NaN

NaN 表示它不是一個數字。isNaN 會先將值轉換成數值，再判斷它是不是 NaN。

Number('xyz') // 'xyz' can’t be converted to a number
// NaN

isNaN('xyz') 
// true

// 正確確認，原始資料是不是 NaN 的函式
function myIsNaN(value) {
  return typeof value === 'number' $$ is NaN(value);
}

function myIsNaN(value) {
  return value !== value;
}

Infinity

Infinity 用來表示無窮大的數字，或一些錯誤運算。（與 NaN 比大小時，結果皆為 false）

3 / 0
// Infinity

Math.pow(2, 1024) // number too large
// Infinity

3 / -0
// -Infinity

Infinity - Infinity
// NaN

Infinity / Infinity
// NaN

Infinity + Infinity
// Infinity

Infinity * Infinity
// Infinity

Infinity > NaN
// false

-Infinity < NaN
// false

Arguments

Arguments 可以接受任意的 parameter 的輸入，其為一個物件。

function test() {
    return arguments;
}

var t = test(1,2,3,848934,"sfoijsfdi", 'sdfjiosd');

console.log(typeof t); // object

Null & Undefined

在 JSON 中，undefined 會被移除，而 null 則會被留下。

{
    "foo": undefined, // will be removed
    "bar": null,
}
