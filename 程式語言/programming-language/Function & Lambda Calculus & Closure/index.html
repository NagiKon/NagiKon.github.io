Function & Lambda Calculus & Closure

發表於2021-09-09|更新於2021-09-11|程式語言

Function

f(x) = y
x 為輸入；y 為輸出

  • 一個 function 一定會有個輸入（argument），並且一定會有個輸出（回傳值）
  • 一個 function 只對收到的參數進行操作
  • 給予一個固定的輸入，一定會對應一個固定的輸出

Referential transparency

概念為：「function 不應該改變外在環境的變數」。

如下面的例子就不符合該概念。

let externalVariable = 'yeah'
let fn = () => externalVariable = 'no'

Function with thread

當使用的 functional 符合 Referential transparency 時，多執行緒不需要去做同步鎖。因為 function 只依賴輸入的值，而不是全域變數。因此就算全域變數被其他執行緒修改，也不會影響到當前執行緒正在使用的 function 的值。所有執行緒都可以對應自己不同的值。

let global = "something"
let function1 = (input) => {
  // works on input
  // changes global
  global = "somethingElse"
}

let function2 = () => {
  if (global === "something") {
    //business logic
  }
}
---
let function1 = (input,global) => {
  // works on input
  // changes global
  global = "somethingElse"
}
let function2 = (global) => {
  if (global === "something") {
      //business logic
    }
}

上面的 function1 被呼叫時，會修改 global 的值，造成其他執行緒在呼叫 function2 時出現錯誤。而下面的 function1 符合 Referential transpaerency 不會讓 function1 影響到 function2。

Lambda 運算式

lambda 運算式只做兩件事情，其本身就是個匿名函數 (anonymous function)。

運算式：$\lambda x.x + 2$
$\blacktriangledown$
數學式子：$f(x) = x + 2$

  1. Application：指派 value 到其他運算式中
  2. Abstraction：定義一個運算式

lambda 運算式類型

$\lambda x.x/y + 2$
x 被綁定在 lambda 運算式中為 約定變數 (bound variable)
y 不知位於何處，被稱為 自由變數 (free variable)，通常其值來自外在執行環境

  1. Close：運算式中，所有的變數都是約定變數
  2. Open：運算式中，存在自由變數

Closure

可以藉由外在執行環境，將上述的自由變數賦予其數值，將其轉換成約定變數(也就是已知的變數)。
將 open 的運算式，轉換成 close 的運算式。這個關閉 open 運算式的動作就叫做 closure。

def f(x):
    def g(y):
        return x + y
    return g  # Return a closure.

def h(x):
    return lambda y: x + y  # Return a closure.

# Assigning specific closures to variables.
a = f(1)
b = h(1)

# Using the closures stored in variables.
assert a(5) == 6
assert b(5) == 6

# Using closures without binding them to variables first.
assert f(1)(5) == 6  # f(1) is the closure.
assert h(1)(5) == 6  # h(1) is the closure.

Clousre 所能 access 到的 scopes：

  1. 宣告範圍內的所有 variables。
  2. Global variables。
  3. outer function’s variables。

下列的程式碼，符合規則 1：

function outer() {
    function inner() {
        let a = 5;
        console.log(a);
    }
    inner();
}

outer(); // 5

下列程式碼，符合規則 2：

let global = 'global';

function outer() {
    function inner() {
        let a = 5;
        console.log(global);
    }
    inner();
}

outer(); // global
